## Free to Use — Not Free Support

Guest Sponsor Info for Microsoft Entra B2B is free software licensed under
[PolyForm Shield 1.0.0](https://polyformproject.org/licenses/shield/1.0.0/).
You can deploy and use it at no cost.

No dedicated support is included with the free distribution.

---

## Community Support (GitHub Issues)

If something does not work as expected, you are welcome to open an issue on
GitHub. Issues are triaged and reviewed on a **best-effort basis** — there is
no guaranteed response time or service-level agreement.

Before opening an issue, please:

- Review the [Setup Guide](/en/setup/)
  for common configuration steps
- Search
  [existing issues](https://github.com/workoho/spfx-guest-sponsor-info/issues)
  for known problems and workarounds

If you have confirmed a bug or reproducible unexpected behaviour:

[Open a GitHub Issue](https://github.com/workoho/spfx-guest-sponsor-info/issues/new){: .btn .btn-secondary}
