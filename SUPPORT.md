# Support

## This project does not provide free support

**Guest Sponsor Info** is a volunteer-maintained open-source project.
GitHub Issues in this repository are reserved exclusively for **bug reports**
and **feature requests** — not for usage questions, configuration help,
deployment assistance, or general "how does this work" inquiries.

Issues opened for support questions will be closed without response.

## Commercial support

If your organisation requires assisted deployment, custom development, or
ongoing support, please contact **Workoho GmbH** directly:

- [workoho.com](https://workoho.com)
